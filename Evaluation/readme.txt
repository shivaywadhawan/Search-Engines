3)In the absence of any retrieved documents, precision is undefined.The formula for precision is dividing the number of relevant documents retrieved by the total number of retrieved documents.
If there are no retrieved documents means denominator is zero and precision is undefined.AP is calculated by summing precision values at each relevant rank position. 
Without any retrieved documents, precision at every rank is undefined, leading to an undefined AP.If a relevant document never gets retrieved, we can assume
the precision corresponding to that relevant doc and in turn the AP to be zero.
